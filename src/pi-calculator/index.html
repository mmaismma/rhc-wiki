What's this all about?

An elegant UI to r/FIUHC's PI calculator spreadsheet.

FIUs (Fédération Internationale de l'Ultra Hardcore) are Formula 1-inspired fan-made championships that run alongside Mindcrack and Hermitcraft UHCs. These chamipinships compete the players against each other based on the gear they are holding. Each player's inventory is noted at certain intervals and a formula generates a single number which shows the player's readiness to combat. This webpage can be used to generate that number.
